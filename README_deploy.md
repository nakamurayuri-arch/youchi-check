# 用地チェック（緯度経度だけ・全項目）— 無料ホストで公開してURL共有

会社情報は使わず、緯度経度だけで用地の全項目を判定します。公開情報のみなので、無料の公開ホスティングに置いてOK（URLを配れば、同僚はインストール不要で開くだけ）。

## 出せる項目
- すぐ動く（キー不要・設置後すぐ）：ハザード（洪水/土砂/津波/高潮）、標高・傾斜、最寄り道路、最寄り建物（民家目安）、海岸距離（重塩害）
- 国土数値情報SHPを置くと自動化：市街化調整区域・用途地域・自然公園・砂防・地すべり・急傾斜・農地（青地/白地・第何種の推定）
- 常にリンク確認：埋蔵文化財（全国統一データが無いため。自治体教委）

## 無料ホストに置く（例：Streamlit Community Cloud・無料）
1. GitHub（無料）にこのフォルダ一式をアップロード（新規リポジトリを作成 → ファイルをドラッグ）。
2. https://share.streamlit.io/ に GitHub でログイン → 「New app」→ そのリポジトリと `app.py` を選択 → Deploy。
3. 数分で `https://xxxx.streamlit.app` のURLが発行される。これを同僚に配るだけ。
   - `...streamlit.app/?lat=33.0598671&lon=131.9332333` の形で座標付きリンクも作れます。
   - 公開URLです。限定したい場合は Streamlit の設定で招待メール制限も可能。

（別案：Hugging Face Spaces も無料。Space作成→Streamlit SDK→ブラウザでファイルをアップロード→自動でURL。GitHub不要。）

## 市街化調整・自然公園・砂防・農地 を自動化する（任意）
国土数値情報SHPを `ksj/` 直下に置くと、上記が自動判定になります（対象県のみでOK）。
DL元 https://nlftp.mlit.go.jp/ksj/

| フォルダ | データ名 |
|---|---|
| ksj/shigaika/ | 都市地域/用途地域(A29)（区域区分＝市街化区域・調整区域） |
| ksj/youto/ | 用途地域(A29) |
| ksj/shizen_kouen/ | 自然公園地域(A10) |
| ksj/sabo/ | 砂防指定地 |
| ksj/jisuberi/ | 地すべり防止区域 |
| ksj/kyukeisha/ | 急傾斜地崩壊危険区域 |
| ksj/nochi/ | 農業地域(A12)（青地/白地） |

置かない項目は「データ未設置 → リンクで確認」と表示され、公式マップへのリンクが出ます。

## うまくデプロイできない場合
- `geopandas` のインストールで失敗したら、`requirements.txt` から `geopandas` `shapely` `pyproj` の3行を消して再デプロイ。
  （市街化調整・自然公園・砂防・農地はリンク確認になりますが、他の項目はそのまま動きます。）

## 注意
- 公開データからの一次確認です。確定は各自治体・現地・公式マップで。
- reinfolib（不動産情報ライブラリ）APIを使えば、市街化調整・用途地域をSHPなしでも取得できます（無料キー・申請5営業日）。必要なら site_checks に組み込み可能。
